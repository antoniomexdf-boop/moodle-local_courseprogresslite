# Guia de Publicacion en GitHub (ES)

## Repositorio

- `moodle-local_courseprogresslite`

## Pasos

1. Verifica `version.php` (release/version).
2. Verifica documentación en `docs/`.
3. Valida instalación/actualización en Moodle 4.1+.
4. Verifica comportamiento de `Habilitar plugin` y `Texto del encabezado`.
5. Crea tag y release.
6. Adjunta el zip del plugin.
