# GitHub Publish Guide (EN)

## Repository

- `moodle-local_courseprogresslite`

## Steps

1. Confirm `version.php` release/version.
2. Confirm docs in `docs/` are updated.
3. Validate install/upgrade in Moodle 4.1+.
4. Verify `Enable plugin` and `Header text` behavior.
5. Create tag and release.
6. Attach plugin zip.
